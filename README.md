# Gym Tracker PPL

Eine einfache iPhone-freundliche Gym-Tracker-Webseite für Push/Pull/Legs.

## GitHub Pages veröffentlichen

1. Neues GitHub Repository erstellen, z.B. `gym-tracker`.
2. Diese Dateien hochladen:
   - `index.html`
   - `README.md`
3. In GitHub öffnen: **Settings → Pages**.
4. Bei **Build and deployment** auswählen:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Auf **Save** klicken.
6. Nach 1–2 Minuten bekommst du einen Link.

## iPhone

Den GitHub-Pages-Link in Safari öffnen → Teilen → Zum Home-Bildschirm.
